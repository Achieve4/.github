# 캡스톤 디자인 웹 개발 ⭐️

### 협업 시 주의사항

깃허브 협업으로 프론트엔드 및 백엔드를 개발하는 경우 서로 다른 레포지토리를 작업해야 합니다 <br />
같은 레포지토리를 만지는 경우 먼저 업로드 한 사람에게 우선순위가 부여되고 그 내용이 반영되며 <br />
이후에 업로드 하는 내용에 대해 깃허브 푸쉬가 막히게 됩니다 <br />
주의해야 합니다!! <br />

<br />
